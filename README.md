# Меню на неделю\n\nЛичный планировщик рецептов, недельного меню, ингредиентов и списка покупок.\n\n## Что уже есть\n\n- каталог из 30 рецептов с карточками и детальными рецептами;\n- недельная сетка на 7 дней и 4 слота питания;\n- режимы получателей «Оба», «Я» и «Алина (Киси)»;\n- локальное сохранение меню в браузере;\n- автоматическая агрегация ингредиентов с категориями и ориентиром по сроку хранения;\n- список покупок с отметками «куплено» и «есть дома»;\n- экспорт состояния в JSON.\n\n## Запуск\n\nПроект не требует сборщика: открой index.html локально или опубликуй содержимое репозитория как статический сайт.\n\n## Дальнейший план\n\n1. Сверить и дополнить карточки рецептов и ингредиенты.\n2. Добавить импорт/редактирование рецептов.\n3. Подключить Supabase для авторизации и синхронизации между устройствами.\n4. Настроить деплой и проверить мобильный сценарий списка покупок.\n\nПищевая информация в карточках обозначена как справочная и не используется для персональных целей или ограничений.\n
